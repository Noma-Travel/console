# Console - Frontend

This React-based UI dynamically loads extensions and gives the user.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

The console will be available at `http://localhost:5174`

## Configuration

Create a `.env` file:

```bash
VITE_API_URL=http://localhost:5000
VITE_DEV_MODE=true
```
